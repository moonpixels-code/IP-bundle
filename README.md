# IP Locator Tool 🌐

	--A simple Python tool that retrieves public IP information such as --
- Country
- Region
- City estimate
- ISP
- Approximate coordinates

	## Installation ##

Python 3 required.

Run:

python iplocate.py

	## Usage ##

Enter a public IP address:

	<IP>, <IP2>

	## Disclaimer ##
	
This tool only provides approximate IP geolocation.
It does not reveal exact addresses.

# DNS Locator Tool 🌐

	--A simple Python tool that retrieves public DNS information such as --
- Domain
- IP

	## Installation ##

Python 3 required.

Run:

python dns_tool.py

	## Usage ##

Enter a public DNS:

	<websitename>.co<country>
